10. NEXT TO BUILD — PRIORITY ORDER
=============================================================================

── TIER 1 — HIGH IMPACT, DATA IN HAND ─────────────────────────────────────

[ ] DWELLING UNIT SERVICE SIZING WIZARD (Art. 120 / 2026, Art. 220 / 2023)
    - THE #1 residential feature. Every panel upgrade, every new
      construction permit needs this.
    - Step-by-step: sq ft → general lighting (2 VA/ft² 2026, 3 VA 2023)
      → small appliance circuits (1500 VA × 2 min) → laundry →
      range/oven demand table → dryer demand table → HVAC (largest wins)
      → demand factors → minimum service amps + conductor + GEC size
    - Standard method AND optional method (120.82 / 220.82)
    - Output: minimum service amps, conductor size, GEC size (Table 250.66)
    - 2026 optional method: first 8 kVA @ 100% (was 10 kVA in 2023)
    - Residential guys buy the app for this feature alone

[ ] GFCI / AFCI REQUIREMENT LOOKUP (210.8, 210.12)
    - Input: location/room type + code year
    - Output: GFCI required? AFCI required? Both? Neither?
    - 2026 adds GFCI for ranges, ovens, dryers, dishwashers, sump pumps
      (210.8(D)) — major change most guys don't know
    - Common inspection fail. Quick win.

[ ] EV CHARGER CIRCUIT SIZING (Art. 625)
    - Input: EVSE nameplate amps + voltage
    - Output: min circuit ampacity (nameplate × 125% — continuous load),
      standard breaker size, minimum wire size, dedicated circuit flag
    - Covers Level 1 / Level 2 / DCFC
    - Hot market right now

[ ] GROUNDING ELECTRODE CONDUCTOR (Table 250.66)
    - Input: largest ungrounded service conductor
    - Output: minimum GEC size (copper and aluminum)
    - Needed by service sizing wizard anyway

[ ] EQUIPMENT GROUNDING CONDUCTOR (Table 250.122)
    - Input: OCPD rating
    - Output: minimum EGC size (copper and aluminum)
    - Quick reference, used constantly

── TIER 2 — SOLID ADDITIONS ────────────────────────────────────────────────

[ ] BURIAL DEPTH LOOKUP (Table 300.7(A) in 2026, Table 300.5 in 2023)
    - Input: wiring method + circuit type
    - Output: minimum cover depth in inches
    - Residential guys trench constantly: pools, sheds, landscape, hot tubs
    - Simple lookup, used all the time

[ ] RECEPTACLE SPACING (210.52)
    - Input: room dimensions
    - Output: minimum receptacle count + max spacing (12 ft rule)
    - Countertop rules (210.52(C)): island, peninsula, wall counter
    - Visual layout would be a differentiator

[ ] RESIDENTIAL CIRCUIT QUICK REFERENCE
    - Kitchen: two 20A small appliance, dedicated dishwasher, disposal,
      refrigerator (dedicated per 2026)
    - Bathroom: dedicated 20A, GFCI, which bathrooms can share
    - Laundry: dedicated 20A
    - One-screen cheat sheet. Apprentices and homeowners love this.

[ ] RACEWAY DERATING QUICK-CHECK
    - Input: conductor count
    - Output: CCC adjustment factor from Table 310.15(C)(1)
    - Fast lookup, currently buried in the ampacity calculator

[ ] GENERATOR SIZING
    - Input: loads to back up → minimum generator kW
    - Transfer switch requirements reference
    - Hot market: storms, grid instability, whole-home backup

12. QA TESTING — VERIFY EVERY NUMBER
=============================================================================

Your app's reputation lives or dies on accuracy. One wrong ampacity
value = 1-star review = ranking death. Test once, test right.

SOURCE OF TRUTH: Your physical 2026 code book.

TEST CASES PER CALCULATOR (5 each):
  1. Normal case — typical everyday values
  2. Edge case low — smallest wire, shortest run
  3. Edge case high — largest wire, longest run
  4. Boundary case — right at a threshold where answer changes
  5. Known gotcha — the tricky NEC exception that trips people up

PRIORITY SPOT CHECKS:
  - Ampacity: Copper #3 THHN 2026 = 115A (2023 = 110A) ← verify this branching
  - Ampacity: Copper #1 THHN 2026 = 145A (2023 = 150A) ← verify
  - Ampacity: Aluminum #8 THHN 2026 = 35A 60°C (2023 = 30A) ← verify
  - Service sizing: 2026 optional method first block = 8 kVA (2023 = 10 kVA)
  - GFCI: 2026 outdoor outlets up to 60A require GFCI (2023 = 50A)
